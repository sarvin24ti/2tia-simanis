🍬 SIMANIS (Sistem Informasi Manajemen Aneka Dodol Siak)

SIMANIS adalah sebuah Sistem Informasi Manajemen berbasis web yang dirancang khusus untuk membantu mengelola operasional produksi, inventaris, dan penjualan usaha Aneka Dodol khas Kabupaten Siak. Aplikasi ini bertujuan untuk mendigitalisasi proses bisnis lokal, meningkatkan efisiensi, dan memudahkan pelacakan data usaha secara terpadu.

✨ Fitur Utama

📦 Manajemen Inventaris: Pelacakan stok bahan baku dan produk dodol siap jual secara real-time.

📊 Laporan & Analisis: Pembuatan laporan penjualan harian, bulanan, dan tahunan beserta grafik yang mudah dipahami.

👥 Manajemen Pengguna/Pegawai: Sistem hak akses (Role-based access control) untuk Admin dan Pemilik Usaha (Owner).
